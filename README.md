# Beam Diffraction Visualizer

This project is designed to visualize the diffraction patterns generated by different types of beams passing through various apertures. The application allows users to adjust beam parameters, aperture parameters, rendering range, and propagation distance. Additionally, users can select the type of beam and aperture from dropdown menus.

## Project Structure

```
beam-diffraction-visualizer
├── main.py
main file
├── README.md
instructions
```

## Features

- **Beam Types**: Supports various beam types including Gaussian beam. Will support various beam types including elliptical Gaussian beam, and Laguerre-Gaussian beam.
